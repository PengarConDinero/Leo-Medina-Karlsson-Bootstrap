Element

Ett html element är liksom en tag. Den definierar din sida och ditt innehåll. Element har en start och slut tag. De brukar också ha en class som ändrar dess egenskap.

HTML

Hypertext Markup Language, Html förkortat, är ett märkspråk för hypertext och är standarden med HTTP för WWW. Därför använder sig alla sidor någon typ av html kod för att ansluta sig till WWW.

ID

Id attributen specificerar en identitet till ett element. Till exempel har du ett h1 (heading) element som du vill ändra fortet men vill inte påverka alla h1 element. Så kan du ge en av dem ett specifikt id som kommer bara ändra h1 med just det id.

Class

Meta taggen definierar metadatan, som är informationen om datorn. < meta> taggen ligger alltid inuti < header> tagen. Till exempel beskriver """ vem som har skrivit sidan, beskrivningen av sidan och hur det ska se ut för andra när det kommer till sidan.

Meta

En responsiv sida är en sida som ändras så att den ser bra ut på alla enheter. Till exempel så ändras storleken på bilderna och texten för mobilen än datorn så det ser bra ut.

Responsive

En responsiv sida är en sida som ändras så att den ser bra ut på alla enheter. Till exempel så ändras storleken på bilderna och texten för mobilen än datorn så det ser bra ut.

Http

WWW är när en webbläsare körs på en online server, vilket ger tillgång för att andra ska söka. http står för Hypertext Transfer Protocol och används för att överföra data från sidor till användaren.

Https

Ifall webbläsaren har ett "s" i slutet av http betyder det att den också är Secure. Då blir det fullständiga namnet Hypertext Transfer Protocol Secure. Vilket indikerar att webbsidan och nätvärket är säkert.

Compressed data

Compressed data är när koden har blivit om kodad på ett sätt som göra att färre informationsbärande enheter behövs användas

CDN

Det står för Content Delivery Network och är gjord för att få din sida att ladda in mycket snabbare.

Local Bootstrap

Det är den lokala bootstrappen man laddar ner till datorn. Alltså den man har i sin visual studio code. Motsatsen skulle vara ifall man var på en webbsida med bootstrap som redan fixad på den.

JQuery

JQuery är de mest populära javascripten. JQuery är gjort för att förenkla koden, och gör det mycket tydligare när man bygger sidor med javascript.

Flexbox

Flexbox är en modell inom HTML som ger dig tillgång att nästintill fritt röra runt lådor med saker på sidan. Detta gör så att man lättare kan göra sidan snyggare och ger den en bättre struktur.

Grid

Grid är också en typ av layout modell som hjälper dig designa din sida. Det är ett form av layout så man slipper elements som exempelvis floats

Code snippets

Code snipping är det man kallar redan färdigjord kod som man specifikt kopierar i delar av sin kod.
